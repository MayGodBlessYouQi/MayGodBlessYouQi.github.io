# 声明变量的方式

# let

let声明的变量只在let代码块中有效

console.log(isLet)   //报错
let isLet = 'aaa';
console.log(isLet)   //aaa

let是块级作用域，函数内部使用let定义后，对函数外部无影响。
let定义的属性不在window上

let a = 1;
function fun(){
    let a = 2;
    console.log(a)  //2
}
fun()
console.log(a)  //1

let的出现解决了for循环嵌套定时器的问题

for(let i=0; i<5; i++){
    setTimeout(()=>{
        console.log(i)
    },1000),
}
// 一秒之后打印 0 1 2 3 4

# const

const定义的变量不可以修改，而且必须初始化

const a;
a = 1; //报错

const b = 1;
b = 2; //报错

const可以用来保存一些用户账号和密码等不易改变的值
const定义变量之后,该变量没有写的权,只有读的权

const username = 'liangyongqi';
const password = '2489919598';
