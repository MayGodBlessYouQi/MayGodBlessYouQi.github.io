    # 数组方法

    # forEach

    forEach便利数组,forEach不会改变原数组的内容,
    里面有三个参数 item 数组的每一项 index 下标 arr 原数组
    每项打印会被执行三次，forEach没有返回值

    const arr = ['张三','李四','王五'];
arr.forEach((item,index,arr) => {
    console.log(item)  //张三 李四 王五
    console.log(index)  //1 2 3
    console.log(arr)   //['张三','李四','王五']
})

    return在forEach中跳出了当前满足条件的循环,循环还在执行

    const num = [8,5,11,45];
num.forEach((item) => {
    console.log(item)  //8 5 11 45
    if(item < 8){
        return;
    }
    console.log(item)  //8 11 45
})

    # map

    map和forEach相比存在返回值 map不改变原数组的内容,
    里面有三个参数 item 数组的每一项 index 下标 arr 原数组
    map会返回一个新的数组

    const arr = [8,11,12,5];
let a = arr.map((item,index,arr) => {
    return item > 10?'大于10':'小于10';
})
console.log(a)  //['小于10','大于10','大于10','小于10']
console.log(arr)  //[8,11,12,5]

    map还可以把数组中的元素转成字符串

    let isArray = [12,{name:'张'},function(){console.log(123)}];
let isStr = isArray.map(String);
console.log(isStr)  //["12", "[object Object]", "function(){console.log(123)}"]

    # filter

    filter过滤数组中不满足条件的元素,把满足条件的元素放到新的数组中,不会改变原数组

    let array = [1, 2, 3, 4, 5];
let temp = array.filter((item, index, arr) => {
    return item > 3;    
});
console.log(temp);    //[4,5]
console.log(array);   //[1,2,3,4,5]

    # reduce

    reduce 接收一个函数作为累加器, 数组中每个值(从左到右)开始缩减, 最终为一个值.
    reduce有两个参数 函数和初始值 初始值可以不写
    函数有四个参数 上一次的值 当前的值 索引 数组

    let array = [1, 2, 3, 4, 5];
let total = array.reduce((a, b) => {
    return a + b;
});
console.log(total);     // 15
