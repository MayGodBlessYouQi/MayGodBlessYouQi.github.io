# 模板字符串

在ES6之前把字符串连接起来的方法是+或者concat()

var a = "上帝";
var b = "是男孩";
console.log(a+b)  //上帝是男孩
console.log(a.concat(b)) //上帝是男孩

模板字面量本质上是包含嵌入式表达式的字符串字面量

let a = '我是div的内容';
let b = `
        <div>${a}</div>
        `;
console.log(b) //<div>我是div的内容</div>
