# node基础

  1. 在桌面创建一个文件夹,先初始化文件夹
    会自动生成一个package.json的配置文件
npm init -y
  1. 创建一个app.js文件
touch app.js
  1. 写app.js的配置文件
const http = require('http');
//引入node里面的http模块
http.createServer((req,res)=>{
    res.end('holl,liang')
    //响应结果,如果没有网页会卡顿
}).listen(8080,function(){
    //listen设置端口
    console.log('8080端口正常');
    //检测端口是否正常
})
  1. 配置package.json文件
"script":{
    "dev":"node app.js"   //npm run dev
}
  1. 启动项目 在github中启动
npm run dev
  1. 报错:::8080可能是端口被占用,更换端口或者解除占用
//端口被占用
use :::8080
  1. 下载nodemon 不管修改了前端页面还是后台，都可以不用重启node
npm install nodemon -g

使用 node 模拟监听 app 文件变化

"script":"nodemon app.js"
